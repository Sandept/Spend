# 📖 About The Project

Spend is an intuitive expense-tracking web application designed to bring clarity to your financial habits.

Ever found yourself spending money through GPay, cash, or credit card to buy something, only to completely forget where that money went or why you spent it? Spend is built to solve exactly this problem.

Whenever you make a purchase, simply open the app, note down the amount, select an appropriate category, and add a quick reason/note. It removes the mystery from your daily spending and helps you stay accountable to yourself.

# ✨ Key Features

Multi-Payment Tracking: Log expenses whether paid via GPay, Cash, or Card.

Smart Categorization: Select specific categories for every purchase.

Detailed Notes: Add custom notes and reasons for your spending so you never forget why you bought something.

Persistent Storage: Safely and securely stores all your tracking data using Supabase.
